TXAllConn
=========

__Provides the allconn command.__



Requirements
------------

- [Xcode 5](https://itunes.apple.com/de/app/xcode/id497799835)
- [Textual 4.2](https://github.com/codeux/Textual)


Instructions
------------

1. Build from source.
2. Double-click AllConn.bundle to install it.
3. Restart Textual.
